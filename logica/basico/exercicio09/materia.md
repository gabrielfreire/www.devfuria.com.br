Exercício 9
===

Escreva um algoritmo para mostrar o sucessor e o antecessor de um número qualquer.


### Comentários

Se o  número em questão for 45...

Qual e o sucessor ?

Qual é o seu antecessor?


### Solução

<div class="code">
<h6>PSEUDOCÓDIGO</h6>
<pre><em>INÍCIO</em>

// Inicia as variáveis
numero     = 0
numero_ant = 0
numero_suc = 0

// Efetua a operação
numero     = 45;
numero_ant = numero - 1
numero_suc = numero + 1

// Mostra os resultados obtidos
mostrar numero_suc, numero_ant

<em>FIM</em></pre>
</div>
