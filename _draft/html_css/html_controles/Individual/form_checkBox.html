O formulário a seguir exemplifica o código e a visualização dos elementos (tags) e atributos usados para a criação dos diversos tipos de controle de formulário HTML.

Permitem seleccionar várias opções de uma lista. 

            Se não clicar em nada...
            array
              empty

            Se clicou && NÂO tem os values definidos então...
            array
              'a' => string '' (length=0)
              'b' => string '' (length=0)
              'c' => string '' (length=0)
              'd' => string '' (length=0)

            Se clicou && tem os 'values' definidos então...
            array
              'a' => string 'pri' (length=3)
              'b' => string 'sec' (length=3)
              'c' => string 'ter' (length=3)
              'd' => string 'for' (length=3)
